# Getting Started

This guide gets a first-time user from clone to usable CLI, GUI, and local docs.

## 1. Clone and enter project

```bash
git clone <your-repo-url> SWC-Studio
cd SWC-Studio
```

## 2. Create environment and install

GUI + CLI install:

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install --upgrade pip
pip install -e ".[gui]"
```

CLI-only install:

```bash
pip install -e .
```

## 3. Verify installation

```bash
swctools --help
swctools-gui --help
```

If `swctools` command is not found, activate the correct venv first:

```bash
echo "$VIRTUAL_ENV"
which python
which swctools
```

## 4. Run the application

CLI examples:

```bash
swctools batch split ./data
swctools validation run ./data/single-soma.swc
swctools morphology smart-decimation ./data/single-soma.swc --write
```

GUI:

```bash
swctools-gui
# or
python -m swctools.gui.main
```

## 5. Core workflow (recommended)

1. Open an SWC file.
2. Run validation.
3. Review warnings/failures.
4. Apply editing operations (label edit, auto-labeling, radii clean, simplification).
5. Save/export result and review generated report logs.

## 6. Where outputs and logs are written

Feature reports are generated by shared backend reporting helpers.

Common examples:

- validation: `<input_stem>_validation_report.txt`
- split folder: `<input_folder>/<input_folder>_split/split_report.txt`
- auto typing folder: `<input_folder>/<input_folder>_auto_typing/auto_typing_report.txt`
- radii cleaning: `*_radii_cleaning_report.txt`
- smart decimation: `*_simplification_log.txt`
- morphology edit session: `*_morphology_session_log.txt`

## 7. Build this Sphinx documentation

Install docs dependencies:

```bash
pip install -r docs/requirements.txt
```

Build HTML:

```bash
sphinx-build -b html docs docs/_build/html
```

Open built docs:

- macOS: `open docs/_build/html/index.html`
- Linux: `xdg-open docs/_build/html/index.html`
- Windows (PowerShell): `start docs/_build/html/index.html`

## 8. Troubleshooting quick list

- GUI fails to start: ensure `PySide6`, `pyqtgraph`, `vispy` are installed (`pip install -e ".[gui]"`).
- command not found: activate venv and reinstall editable package.
- plugin not loading: check `PYTHONPATH` and module import path.
- validation or cleaning behavior unexpected: verify feature JSON config under `swctools/tools/*/configs`.
